# La Casa de Sammy 🏡

> Tu segunda casa en el Valle de la Longevidad.

Sitio web oficial de **La Casa de Sammy**, una casa de campo para grupos familiares ubicada en Vilcabamba, Loja — Ecuador.

## 🌄 Sobre

Vilcabamba, conocido como el *Valle de la Longevidad*, es un rincón mágico al sur del Ecuador. La Casa de Sammy ofrece habitaciones rústicas, piscina con vista al valle, huerto propio y experiencias auténticas para familias que viajan juntas.

## 📍 Ubicación

- **Dirección:** Vía Yamburara, sector San José · Vilcabamba · Loja · Ecuador
- **Desde Loja:** 41 km — 1 hora en auto
- **Aeropuerto cercano:** Catamayo (LOH) — 75 km

## ✨ Lo que ofrecemos

- 5 habitaciones para hasta 12 huéspedes
- Piscina con vista al valle
- Huerto propio · desayuno casero
- Cabalgatas, senderismo y aguas termales
- Pet friendly

## 🛠️ Tecnología

Sitio estático construido con HTML + React (vía Babel standalone). Sin dependencias de build — se puede servir desde cualquier hosting estático (GitHub Pages, Netlify, Vercel).

### Estructura

```
├── index.html          # página principal
├── shared.jsx          # componentes compartidos (Photo, ImagePlaceholder)
├── logo.png            # logo de la casa
└── photos/             # fotografías
```

### Ejecutar localmente

Abre `index.html` directamente en el navegador, o sirve la carpeta con cualquier servidor estático:

```bash
python3 -m http.server 8000
# luego abre http://localhost:8000
```
